void backtrack(路径, 选择列表) {
    // 1. 终止条件 (Base Case)
    if (满足结束条件) {
        存放结果;
        return;
    }
    for (选择 : 选择列表) {
        // 2. 剪枝 (Pruning) - 可选
        if (不符合业务逻辑或约束) continue;

        // 3. 做选择

        标记选择;
        将选择添加到路径;

        // 4. 递归进入下一层

        backtrack(路径, 选择列表);

        // 5. 撤销选择 (Backtrack) 
        // 回到之前的状态，为下一次 for 循环尝试别的选择做准备

        移出路径;
        还原标记;
    }
}

括号生成：	左括号不能超过 n；右括号数量不能超过左括号----判断可否加‘（’or'）'
单词搜索：	坐标不能越界；字符必须匹配 word[i]；不能重复使用同一个格子----在二维矩阵中向四个方向探索
分割回文串：	只有当前截取的子串是回文，才继续往下切----尝试不同的切割长度（i 从 start 到 n-1）
N 皇后：	同列、左斜线、右斜线不能有其他皇后----逐行放置，每行通过 for 循环尝试每一列。
